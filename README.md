## 💻 Getting Started <a name="getting-started"></a>

> This use this project, install Ruby on your local machine and then clone the project.


### Prerequisites

In order to run this project you need:

- Ruby
- Visual studio code editor or any other code editor
- Git

### Setup

Clone this repository to your desired folder:

1. cd to your_folder
2. git clone https://github.com/anbehindY/Morse-decoder.git
3. cd Morse-decoder
4. Use an editor to open the code
